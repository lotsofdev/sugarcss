# s-color
<div class="_namespace">@lotsof.sugarcss.css.function</div>
<div class="_metas">
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">0.0.1</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">0.0.1</span></div>
</div>
<div class="_description">
Apply a color from the colors registered ones.
Allow to apply modifiers or shades to the color like darken, lighten, saturate, desaturate, spin, etc...
Here's the list of available modifiers:

- lighten: 0-100
- darken: 0-100
- saturate: 0-100
- desaturate: 0-100
- spin: 0-360
- alpha: 0-1

The shades are registered like so: --s-shade-placeholder: --darken 10;
</div>
<div class="_params">
## Params
1. <span class="_name">color<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
   - The color name you want to apply like "main", "accent", etc...
2. <span class="_name">shadeOrModifiers<span class="_required">*</span></span><span class="_default"></span> <span class="_type">String</span>
   - The shade name you want to apply like "text", "placeholder", etc... or a string of modifiers like --lighten 10 --saturate 20
</div>
<div class="_return">
## Return
- <span class="_description">The generated css</span><span class="_default"></span><span class="_type">Css</span>
</div>
<div class="_examples">
## Examples
```css
--s-shade-test: --lighten 10;

.my-element {
  color: s-color(accent);
  color: s-color(accent, test);
  color: s-color(accent, --darken 20 --spin 30);
}
```
